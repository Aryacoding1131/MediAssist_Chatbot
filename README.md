# MediAssist_Chatbot
This Chatbot is purely built using Groq LLM module for analysis generation, GTTS for text-to-speech, reportlab for document generation. In maps are built on using folium, flask for map generation and math module for to get the nearby hospitals and clinics


### libraries requird 
1. flask, folium - for maps
2. math - for obtaining the near by hospitals and clinics by using live location
3. groq - LLM for text generation from user symptoms
4. reportlab - for pdf of report generation
5. gtts- for TTS
